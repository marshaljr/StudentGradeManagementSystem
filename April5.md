This project is a JavaFX desktop application built with MVC and OOP. The structure is split clearly: the [model package](/Users/marshal/Documents/StudentGradeManagementSystem/src/main/java/com/sms/model) stores the data classes, the [controller package](/Users/marshal/Documents/StudentGradeManagementSystem/src/main/java/com/sms/controller) handles user actions, the [dao package](/Users/marshal/Documents/StudentGradeManagementSystem/src/main/java/com/sms/dao) handles database access, the [util package](/Users/marshal/Documents/StudentGradeManagementSystem/src/main/java/com/sms/util) contains shared helpers, and the [FXML views](/Users/marshal/Documents/StudentGradeManagementSystem/src/main/resources/fxml) define the screens.

OOP is used in these ways: encapsulation appears in [UserAccount.java](/Users/marshal/Documents/StudentGradeManagementSystem/src/main/java/com/sms/model/UserAccount.java#L16) because the fields are private and accessed through methods; abstraction is shown by the abstract class and abstract methods in [UserAccount.java](/Users/marshal/Documents/StudentGradeManagementSystem/src/main/java/com/sms/model/UserAccount.java#L13); inheritance is used because [AdminUser.java](/Users/marshal/Documents/StudentGradeManagementSystem/src/main/java/com/sms/model/AdminUser.java#L1), [TeacherUser.java](/Users/marshal/Documents/StudentGradeManagementSystem/src/main/java/com/sms/model/TeacherUser.java#L10), and [StudentUser.java](/Users/marshal/Documents/StudentGradeManagementSystem/src/main/java/com/sms/model/StudentUser.java#L10) all extend UserAccount; and polymorphism is used when [LoginController.java](/Users/marshal/Documents/StudentGradeManagementSystem/src/main/java/com/sms/controller/LoginController.java#L135) calls `user.getDashboardPath()`, which opens different dashboards depending on the actual object type, such as [TeacherUser.java](/Users/marshal/Documents/StudentGradeManagementSystem/src/main/java/com/sms/model/TeacherUser.java#L33) and [StudentUser.java](/Users/marshal/Documents/StudentGradeManagementSystem/src/main/java/com/sms/model/StudentUser.java#L40).

JavaFX is used for the UI layer. [MainApp.java](/Users/marshal/Documents/StudentGradeManagementSystem/src/main/java/com/sms/MainApp.java#L22) starts the app and loads [LoginView.fxml](/Users/marshal/Documents/StudentGradeManagementSystem/src/main/resources/fxml/LoginView.fxml#L8). That FXML file binds to [LoginController.java](/Users/marshal/Documents/StudentGradeManagementSystem/src/main/java/com/sms/controller/LoginController.java#L26), while [AdminDashboard.fxml](/Users/marshal/Documents/StudentGradeManagementSystem/src/main/resources/fxml/AdminDashboard.fxml#L8) binds to [AdminDashboardController.java](/Users/marshal/Documents/StudentGradeManagementSystem/src/main/java/com/sms/controller/AdminDashboardController.java#L26). The module setup in [module-info.java](/Users/marshal/Documents/StudentGradeManagementSystem/src/main/java/module-info.java#L1) opens the packages to JavaFX so FXML injection works correctly.
